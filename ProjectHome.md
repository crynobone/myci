Ext.CodeIgniter consist of enhancement made to CodeIgniter PHP Framework which many programmer would find useful in most of the time.
### Features ###
  * [Template Library](http://crynobone.com/template-library-for-ext-codeigniter/)
  * [Option Library](http://crynobone.com/option-library-for-ext-codeigniter/)
  * Authentication Library
### Extends The Core ###
  * Improved Table Library
  * Smart URI Segmentation
  * Multiple helpers
